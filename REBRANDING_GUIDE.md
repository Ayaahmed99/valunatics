# Rebranding Guide - Brand Color Palette

## Brand Colors

The dashboard has been rebranded with the following color palette:

- **Green**: `#0ec17c` - Primary action color, positive indicators
- **Violet Light**: `#2a008d` - Secondary accent, gradients
- **Violet Dark**: `#170454` - Backgrounds, cards
- **Dark Background**: `#040015` - Main background

## Tailwind Configuration

Colors are available in Tailwind as:
- `bg-brand-green`
- `bg-brand-violetLight`
- `bg-brand-violetDark`
- `bg-brand-dark`
- `text-brand-green`
- `border-brand-green`
- etc.

## Design Principles

1. **Dark Theme**: All components use dark background (`bg-brand-dark`)
2. **Glass Morphism**: Cards use backdrop blur with semi-transparent backgrounds
3. **Gradients**: Primary buttons use green-to-violet gradients
4. **Hover Effects**: Interactive elements have smooth transitions and scale effects
5. **Color Coding**: 
   - Green = Positive, success, actions
   - Violet = Secondary, information, backgrounds
   - White/Transparent = Text, borders, overlays

## Component Updates Needed

All components should use:
- Dark backgrounds (`bg-brand-dark`)
- Brand-colored buttons and accents
- White/transparent text for readability
- Glass morphism cards with borders
- Smooth hover transitions
